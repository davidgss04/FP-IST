# FP - Foundations of Programming

The two projects made during the academic year 2022/23 of Foundations of Programming at Instituto Superior Técnico.

## First Project

The first project consists in three independent tasks: 
    - Text parsing 
    - Calculating mandates using Hondt's method
    - Solving simple systems of equations

- Click [here](https://github.com/davidgss04/FP-IST/blob/main/Project1/FP2223_P1_220929.pdf) to check the the project sheet
- Click [here](https://github.com/davidgss04/FP-IST/blob/main/Project1/proj.py) to check the project code

## Second Project

The second project consists in a console based Minesweeper gameplay.

- Click [here](https://github.com/davidgss04/FP-IST/blob/main/Project2/FP2223_P2_221031.pdf) to check the the project sheet
- Click [here](https://github.com/davidgss04/FP-IST/blob/main/Project2/proj.py) to check the project code

In order to play the game you will need to head to the code's directory and run the code file:
```
cd Project2 #Assuming you're on the root directory
python3 proj.py
```

## Tests

In order two run the given tests you will need pytest which can be installed like this: 

```
pip install pytest
```

- For the first project, in order to run the tests, and assuming you're in the root directory:

```
cd Project1/tests
pytest
```

- And for the second project:

```
cd Project2/tests
pytest
```
